# JSON Parser

一个简洁、现代、功能强大的在线JSON解析工具，采用macOS风格设计。

[点此访问](https://tingwdsj.github.io/json_parser.github.io/json-parser.html)

## ✨ 特性

### 核心功能

- 🔄 **实时解析** - 输入时自动检测并解析JSON格式
- 🎨 **语法高亮** - 不同数据类型使用不同颜色区分
- ✅ **格式验证** - 实时显示JSON合法性及错误位置
- 📊 **统计信息** - 显示字符数、行数、解析时间

### 编辑功能

- 🔢 **行号显示** - 支持输入和输出区域的行号显示/隐藏
- 📋 **一键复制** - 快速复制解析结果到剪贴板
- 💾 **导出文件** - 将解析结果导出为.json文件
- 🗑️ **清空内容** - 一键清空所有输入

### JSON操作

- 📖 **展开/折叠** - 支持单个节点和全部节点的展开折叠
- 🎭 **转义控制** - 保留/取消转义字符显示
- ✨ **格式化** - 美化JSON输出（带缩进）
- 📦 **压缩** - 压缩JSON为单行

### 界面功能

- 🌓 **主题切换** - 深色/浅色编辑器主题
- 👁️ **输入控制** - 隐藏/显示原始输入面板
- ⛶ **全屏模式** - 全屏查看解析结果
- 📱 **响应式设计** - 自适应桌面和移动设备

## 🎯 使用方法

### 基本使用

1. **打开工具**
   
   - 直接在浏览器中打开 `json-parser.html` 文件
   - 无需安装，无需服务器，纯静态文件

2. **输入JSON**
   
   - 在左侧"输入"面板粘贴或输入JSON文本
   - 右侧"结果"面板会实时显示解析结果

3. **查看结果**
   
   - 合法的JSON会显示语法高亮的解析结果
   - 非法的JSON会显示详细的错误信息

### 高级操作

#### 展开/折叠

- 点击节点前的 ▼ 图标折叠内容
- 点击 ▶ 图标展开内容
- 使用工具栏的"展开全部"/"折叠全部"按钮批量操作

#### 格式化与压缩

- **格式化**：点击工具栏的格式化按钮，输出多行带缩进的JSON
- **压缩**：再次点击该按钮，输出单行紧凑的JSON
- 操作针对解析结果，不影响原始输入

#### 转义控制

- **显示转义**：点击"显示转义"按钮，字符串中的特殊字符显示为转义符号（如 `\n`）
- **隐藏转义**：再次点击，特殊字符显示为实际字符

#### 行号显示

- 点击工具栏的"行号"按钮
- 同时显示输入和输出区域的行号
- 便于定位和对照

#### 主题切换

- 点击工具栏的主题切换按钮（太阳图标）
- **深色主题**（默认）：深色背景，适合长时间使用
- **浅色主题**：浅色背景，适合明亮环境

#### 隐藏输入面板

- 点击左侧面板顶部的向上箭头按钮隐藏输入框
- 点击右侧面板顶部的向左箭头按钮显示输入框
- 隐藏后解析结果占据整个显示区域

## 🎨 界面预览

### 状态指示

| 状态   | 颜色  | 说明       |
| ---- | --- | -------- |
| 等待输入 | 灰色  | 尚未输入内容   |
| 格式正确 | 绿色  | JSON格式合法 |
| 格式错误 | 红色  | JSON格式有误 |

| 类型            | 颜色（深色主题）     | 颜色（浅色主题）     |
| ------------- | ------------ | ------------ |
| 键名 (key)      | 蓝色 `#4fc1ff` | 蓝色 `#0066cc` |

| 数字 (number)   | 绿色 `#b5cea8` | 绿色 `#098658` |
| 布尔值 (boolean) | 蓝色 `#569cd6` | 蓝色 `#0000ff` |
| null          | 灰色 `#808080` | 灰色 `#808080` |
| 括号            | 黄色 `#ffd700` | 黄色 `#ffd700` |
